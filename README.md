# egovWebWatcher
산학 프로젝트 "웹 어플리케이션 테스팅 자동화 서비스를 제공하는 웹 어플리케이션(Web Watcher)"를 표준프레임워크로 이전하는 프로젝트
표준 프레임워크 3.8 버전 업그레이드 이후 공통컴포넌트 적용을 위해선 프로젝트 구조 변화로 인해 프로젝트 구조 변경 필요
이에 따라 프로젝트 이전
